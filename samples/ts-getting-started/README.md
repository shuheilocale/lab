## Notes

install ts compiler globally

```bash
npm install -g typescript
tsc --v
```

how to run

```bash
# compile > main.js
tsc main.ts
# run
node main.js
# out>Hello, World!
```


```bash
# compile and run (a little bit slower than above pattern)
ts-node main.ts
# out>Hello, World!
```
